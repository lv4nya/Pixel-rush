# Security Policy

Pixel Brew Bakery is a small local game prototype. It does not currently handle accounts, payments, or user data.

## Reporting Issues

If you find a security issue, please avoid posting exploit details publicly. Open a private report through your repository host if available, or contact the repository owner directly.

## Dependencies

Keep dependencies current with:

```bash
npm install
npm run build
```

Review dependency updates before merging them.
